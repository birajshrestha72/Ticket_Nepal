# Frontend (Feature-Wise Starter)

Simple React app to build features one by one.

## Run

```bash
npm install
npm run dev
```

By default, frontend calls backend at `http://127.0.0.1:8000`.

If needed, set:

```bash
VITE_API_BASE_URL=http://127.0.0.1:8000
```

## Starter feature folders

- `src/features/status`
- `src/features/auth`
- `src/features/buses`
- `src/features/bookings`
