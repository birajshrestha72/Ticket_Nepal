# Bus Images

Place bus images in this directory with the following naming convention:

## Required Image Names:
- `ac.jpg` - For AC buses
- `non_ac.jpg` - For Non-AC buses  
- `deluxe.jpg` - For Deluxe buses
- `semi_deluxe.jpg` - For Semi-Deluxe buses
- `sleeper.jpg` - For Sleeper buses
- `seater.jpg` - For Seater buses
- `default-bus.jpg` - Fallback image for all bus types

## Image Specifications:
- **Format**: JPG, PNG, or WebP
- **Recommended size**: 800x600px or 4:3 aspect ratio
- **File size**: Keep under 500KB for optimal loading

## Notes:
- Images should be high-quality photos of actual buses
- Ensure images are properly licensed for use
- The system will automatically fall back to `default-bus.jpg` if a specific bus type image is not found
